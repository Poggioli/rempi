## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/slider
```

## Uso

Importe o <strong>Slider</strong> e junte todas as partes do componente.

```jsx
import { Slider } from "@rempi-ui/slider";

() => (
  <Slider.Root defaultValue={[50]} max={100} step={1}>
    <Slider.Thumb aria-label="Volume" />
  </Slider.Root>
);
```

## Sobre

Rempi-UI Slider é baseado no [Radix Slider](https://www.radix-ui.com/primitives/docs/components/slider).

## API reference

### Root

Contém todas as partes do controle slider. Ele renderizará um elemento `input` para cada controle quando usado dentro de um `form` para garantir que os eventos se propaguem corretamente.
