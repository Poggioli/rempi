## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/alert-dialog
```

## Uso

Importe o <strong>AlertDialog</strong> e junte todas as partes do componente.

```jsx
import { AlertDialog } from "@rempi-ui/alert-dialog";
import { Button } from "@rempi-ui/button";

() => (
  <AlertDialog.Root>
    <AlertDialog.Trigger>
      <Button color="error">Excluir conta.</Button>
    </AlertDialog.Trigger>
    <AlertDialog.Portal>
      <AlertDialog.Overlay />
      <AlertDialog.Content>
        <AlertDialog.Title>Você tem certeza absoluta?</AlertDialog.Title>
        <AlertDialog.Description>
          Essa ação não pode ser desfeita. Isso excluirá permanentemente sua
          conta e removará seus dados de nossos servidores.
        </AlertDialog.Description>
        <AlertDialog.Footer>
          <AlertDialog.Cancel>
            <Button variant="text">Cancelar</Button>
          </AlertDialog.Cancel>
          <AlertDialog.Action>
            <Button color="error">Sim, deletar conta</Button>
          </AlertDialog.Action>
        </AlertDialog.Footer>
      </AlertDialog.Content>
    </AlertDialog.Portal>
  </AlertDialog.Root>
);
```

## Sobre

Rempi-UI Alert Dialog é baseado no [Radix Alert Dialog](https://www.radix-ui.com/primitives/docs/components/alert-dialog).
