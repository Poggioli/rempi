## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/radio
```

## Uso

Importe o <strong>Radio</strong> e junte todas as partes do componente.

```jsx
import { Radio } from "@rempi-ui/radio";
import { Flex } from "@rempi-ui/flex";
import { Heading } from "@rempi-ui/heading";

() => (
  <Flex flexDirection="column">
    <Heading variant="6" fontWeight="normal" fontSize="xlg">
      Selecione sua cor favorita
    </Heading>
    <Radio.Root>
      <Radio.Label>
        <Radio.Item value="green" /> Verde
      </Radio.Label>
      <Radio.Label>
        <Radio.Item value="purple" /> Roxa
      </Radio.Label>
      <Radio.Label>
        <Radio.Item value="brown" /> Marrom
      </Radio.Label>
      <Radio.Label>
        <Radio.Item value="white" /> Branco
      </Radio.Label>
      <Radio.Label>
        <Radio.Item value="yellow" /> Amarelo
      </Radio.Label>
    </Radio.Root>
  </Flex>
);
```

## Sobre

Rempi-UI Radio é baseado no [Radix Radio](https://www.radix-ui.com/primitives/docs/components/radio-group).

### Item

É uma mistura de [Radix Radio Item](https://www.radix-ui.com/primitives/docs/components/radio-group#item) e [Radix Radio Indicator](https://www.radix-ui.com/primitives/docs/components/radio-group#indicator).<br/><br/>
Um item no grupo que pode ser marcado. Um `input` também será renderizado quando usado dentro de um `form` para garantir que os eventos se propaguem corretamente. É renderizado quando o item de rádio está em estado marcado. Você pode estilizar este elemento diretamente ou usá-lo como um invólucro para colocar um ícone dentro dele, ou ambos.
