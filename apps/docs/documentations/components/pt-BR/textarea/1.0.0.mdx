# Textarea

<Description text="Captura a entrada do usuário." />

<SelectVersion />

<PreviewComponent name="textarea" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/textarea
```

## Uso

Importe o <strong>Textarea</strong> e junte todas as partes do componente.

```jsx
import { Textarea } from "@rempi-ui/textarea";

() => (
  <Textarea.Label alignItems="flex-start" flexDirection="column">
    Mensagem
    <Textarea.Root placeholder="Digite sua mensagem aqui" />
  </Textarea.Label>
);
```
