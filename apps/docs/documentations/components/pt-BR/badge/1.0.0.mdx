## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/badge
```

## Uso

Importe o <strong>Badge</strong>.

```jsx
import { Badge } from "@rempi-ui/badge";

() => <Badge>Olá eu sou um Badge</Badge>;
```
