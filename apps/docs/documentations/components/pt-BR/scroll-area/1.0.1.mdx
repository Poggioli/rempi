## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/scroll-area
```

## Uso

Importe o <strong>ScrollArea</strong> e junte todas as partes do componente.

```jsx
import { Flex } from "@rempi-ui/flex";
import { ScrollArea } from "@rempi-ui/scroll-area";
import { Separator } from "@rempi-ui/separator";
import { Typography } from "@rempi-ui/typography";
import { FC, Fragment } from "react";

const TAGS = Array.from({ length: 50 }).map(
  (_, i, a) => `v1.2.0-beta.${a.length - i}`
);

() => (
  <Flex alignItems="center" justifyContent="center" flexWrap="wrap">
    <ScrollArea.Root>
      <ScrollArea.Viewport>
        <div>
          <Typography fontWeight="regular">Tags</Typography>
          {TAGS.map((tag) => (
            <Fragment key={tag}>
              <Separator />
              <Typography variant="body2">{tag}</Typography>
            </Fragment>
          ))}
        </div>
      </ScrollArea.Viewport>
      <ScrollArea.Scrollbar orientation="vertical">
        <ScrollArea.Thumb />
      </ScrollArea.Scrollbar>
      <ScrollArea.Scrollbar orientation="horizontal">
        <ScrollArea.Thumb />
      </ScrollArea.Scrollbar>
      <ScrollArea.Corner />
    </ScrollArea.Root>
  </Flex>
);
```

## Sobre

Rempi-UI Scroll Area é baseado no [Radix Scroll Area](https://www.radix-ui.com/primitives/docs/components/scroll-area).

Contém todas as partes do scroll area.

<APIReference
  data={[
    {
      name: "type",
      type: '"auto" | "always" | "scroll" | "hover"',
      typeSimple: "enum",

      description: (

          <br />
          <br />
          <code>"auto"</code> significa que as barras de rolagem são visíveis quando
          o conteúdo está transbordando na orientação correspondente.
          <br />
          <code>"always"</code> significa que as barras de rolagem estão sempre visíveis,
          independentemente de o conteúdo estar transbordando ou não.
          <br />
          <code>"scroll"</code> significa que as barras de rolagem são visíveis quando
          o usuário está rolando na orientação correspondente.
          <br />
          <code>"hover"</code> significa que as barras de rolagem são visíveis quando
          o usuário está rolando na orientação correspondente e quando o usuário
          está passando o mouse sobre a área de rolagem.
        </span>
      ),
    },
    {
      name: "scrollHideDelay",
      type: "number",
      defaultValue: "600",
      description: (

          <code>"hover"</code>, esta propriedade determina a duração, em

          usuário parar de interagir com elas.
        </span>

    },
    {
      name: "dir",
      required: false,
