## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/checkbox
```

## Uso

Importe o <strong>Checkbox</strong> e junte todas as partes do componente.

```jsx
import { Checkbox } from "@rempi-ui/checkbox";

() => (
  <Checkbox.Label>
    <Checkbox.Check />
    Li e aceito os termos de uso.
  </Checkbox.Label>
);
```

## Sobre

Rempi-UI Checkbox é baseado no [Radix Checkbox](https://www.radix-ui.com/primitives/docs/components/checkbox).

## API reference

### Check

É uma mistura de [Radix Checkbox Root](https://www.radix-ui.com/primitives/docs/components/checkbox#root) e [Radix Checkbox Indicator](https://www.radix-ui.com/primitives/docs/components/checkbox#indicator).<br/><br/>
Contém todas as partes de um checkbox. Um elemento `input` também será renderizado quando usado dentro de um `formu` para garantir que os eventos se propaguem corretamente. <br /><br />
Renderiza quando o checkbox está marcado ou em um estado indeterminado. Você pode estilizar este elemento diretamente ou usá-lo como um invólucro para inserir um ícone, ou ambos.
