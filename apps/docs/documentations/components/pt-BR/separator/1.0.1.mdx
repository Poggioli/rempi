## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/separator
```

## Uso

Importe o <strong>Separator</strong>.

```jsx
import { Separator } from "@rempi-ui/separator";

() => (
  <Flex justifyContent="center">
    <Heading>Entre nós</Heading>
    <Separator />
    <Heading>é um Separator</Heading>
  </Flex>
);
```

## Sobre

Rempi-UI Separator é baseado no [Radix Separator](https://www.radix-ui.com/primitives/docs/components/separator).
