# Label

<Description text="Renderiza uma Label acessível associado aos controles." />

<SelectVersion />

<PreviewComponent name="label" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/label
```

## Uso

Importe o <strong>Label</strong>.

```jsx
import { Label } from "@rempi-ui/label";

() => <Label>Olá eu sou uma Label</Label>;
```

## Sobre

Rempi-UI Label é baseado no [Radix Label](https://www.radix-ui.com/primitives/docs/components/label).

## API reference

### Label

É uma mistura de [Radix Label](https://www.radix-ui.com/primitives/docs/components/label) e [Rempi-UI Flex](docs/flex). <br /><br />
Contém o conteúdo para a Label. Possui todos os atributos do [Rempi-UI Flex](docs/flex).
