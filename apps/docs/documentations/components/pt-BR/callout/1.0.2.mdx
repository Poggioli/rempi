## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/callout
```

## Uso

Importe o <strong>Callout</strong> e junte todas as partes do componente.

```jsx
import { Callout } from "@rempi-ui/callout";
import { Info } from "lucide-react";

() => (
  <Callout.Root>
    <Callout.Icon>
      <Info size={16} />
    </Callout.Icon>
    <Callout.Text>
      Você precisará de privilégios de administrador para instalar e acessar
      este aplicativo.
    </Callout.Text>
  </Callout.Root>
);
```
