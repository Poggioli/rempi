# Dropdown Menu

<Description text="Apresenta um menu para o usuário, como um conjunto de ações ou funções, acionado por um botão." />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/dropdown-menu
```

## Uso

Importe o <strong>DropdownMenu</strong> e junte todas as partes do componente.

```jsx
import { DropdownMenu } from "@rempi-ui/dropdown-menu";
import { IconButton } from "@rempi-ui/button";
import { SlidersHorizontal } from "lucide-react";

() => (
  <DropdownMenu.Root>
    <DropdownMenu.Trigger>
      <IconButton
        aria-label="Configurações"
        variant="contained"
        shape="rounded"
        color="primary"
      >
        <SlidersHorizontal size={16} />
      </IconButton>
    </DropdownMenu.Trigger>
    <DropdownMenu.Content>
      <DropdownMenu.Item>
        Voltar <div>⌘+[</div>
      </DropdownMenu.Item>
      <DropdownMenu.Item disabled>
        Avançar <div>⌘+]</div>
      </DropdownMenu.Item>
      <DropdownMenu.Item>
        Recarregar <div>⌘+R</div>
      </DropdownMenu.Item>

      <DropdownMenu.SubMenu>
        <DropdownMenu.SubMenuTrigger>
          Mais ferramentas
        </DropdownMenu.SubMenuTrigger>
        <DropdownMenu.SubMenuContent sideOffset={2} alignOffset={-5}>
          <DropdownMenu.Item>
            Salvar página como... <div>⌘+S</div>
          </DropdownMenu.Item>
          <DropdownMenu.Item>Criar atalho...</DropdownMenu.Item>
          <DropdownMenu.Item>Nome da janela...</DropdownMenu.Item>
          <DropdownMenu.Separator />
          <DropdownMenu.Item>Ferramentas do desenvolvedor</DropdownMenu.Item>
        </DropdownMenu.SubMenuContent>
      </DropdownMenu.SubMenu>

      <DropdownMenu.Separator />

      <DropdownMenu.CheckboxItem
        checked={bookmarksChecked}
        onCheckedChange={setBookmarksChecked}
      >
        Mostrar favoritos <div>⌘+B</div>
      </DropdownMenu.CheckboxItem>
      <DropdownMenu.CheckboxItem
        checked={urlsChecked}
        onCheckedChange={setUrlsChecked}
      >
        Mostras URLs completas
      </DropdownMenu.CheckboxItem>

      <DropdownMenu.Separator />

      <DropdownMenu.Label>Pessoas</DropdownMenu.Label>

      <DropdownMenu.RadioGroup value={person} onValueChange={setPerson}>
        <DropdownMenu.RadioItem value="pedro">
          Pedro Duarte
        </DropdownMenu.RadioItem>
        <DropdownMenu.RadioItem value="colm">Colm Tuite</DropdownMenu.RadioItem>
      </DropdownMenu.RadioGroup>
    </DropdownMenu.Content>
  </DropdownMenu.Root>
);
```

## Sobre

<strong>Rempi-UI Dropdown Menu</strong> é baseado no [Radix Dropdown Menu](https://www.radix-ui.com/primitives/docs/components/dropdown-menu).
