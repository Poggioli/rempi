# Button

<Description text="Botões permitem que os usuários tomem ações e façam escolhas com um único toque." />

<SelectVersion />

<PreviewComponent name="button" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/button
```

## Uso

Importe o <strong>Button</strong> ou <strong>IconButton</strong>.

```jsx
import { Button, IconButton } from "@rempi-ui/button";
import { TrashIcon } from "lucide-react";

() => <Button>Olá eu sou um Botão</Button>;
() => (
  <IconButton>
    <TrashIcon />
  </IconButton>
);
```
