## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/toggle
```

## Uso

Importe o <strong>Toggle</strong>.

```jsx
import { Flex } from "@rempi-ui/Flex";
import { Toggle } from "@rempi-ui/toggle";
import { Italic } from "lucide-react";

() => (
  <Flex alignItems="center" justifyContent="center">
    <Toggle>
      <Italic size={16} />
    </Toggle>

    <Toggle defaultPressed>
      <Italic size={16} />
    </Toggle>

    <Toggle disabled>
      <Italic size={16} />
    </Toggle>

    <Toggle disabled defaultPressed>
      <Italic size={16} />
    </Toggle>
  </Flex>
);
```

## Sobre

<strong>Rempi-UI Toggle</strong> é baseado no [Radix Toggle](https://www.radix-ui.com/primitives/docs/components/toggle).
