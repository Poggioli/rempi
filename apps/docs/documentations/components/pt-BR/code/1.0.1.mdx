# Code

<Description text="Marca o texto para indicar um pequeno fragmento de código." />

<SelectVersion />

<PreviewComponent name="code" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/code
```

## Uso

Importe o <strong>Code</strong>.

```jsx
import { Code } from "@rempi-ui/code";

() => <Code>console.log('Olá eu sou um código')</Code>;
```
