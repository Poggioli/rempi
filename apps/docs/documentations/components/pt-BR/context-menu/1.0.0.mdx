# Context Menu

<Description text="Exibe um menu localizado no ponteiro, acionado por um clique com o botão direito ou uma pressão longa." />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/context-menu
```

## Uso

Importe o <strong>ContextMenu</strong> e junte todas as partes do componente.

```jsx
import { ContextMenu } from "@rempi-ui/context-menu";
import { Flex } from "@rempi-ui/flex";
import { Typography } from "@rempi-ui/typography";

() => (
  <ContextMenu.Root>
    <ContextMenu.Trigger>
      <Flex flexDirection="row" alignItems="center" justifyContent="center">
        <Typography>Clique com o botão direito aqui</Typography>
      </Flex>
    </ContextMenu.Trigger>
    <ContextMenu.Content>
      <ContextMenu.Item>
        Voltar
        <div>⌘+[</div>
      </ContextMenu.Item>
      <ContextMenu.Item disabled>
        Avançar
        <div>⌘+]</div>
      </ContextMenu.Item>
      <ContextMenu.Item>
        Recarregar
        <div>⌘+R</div>
      </ContextMenu.Item>

      <ContextMenu.SubMenu>
        <ContextMenu.SubMenuTrigger>
          Mais ferramentas
        </ContextMenu.SubMenuTrigger>
        <ContextMenu.SubMenuContent sideOffset={2} alignOffset={-5}>
          <ContextMenu.Item>
            Salvar página como...
            <div>⌘+S</div>
          </ContextMenu.Item>
          <ContextMenu.Item>Criar um atalho</ContextMenu.Item>
          <ContextMenu.Item>Nome da janela</ContextMenu.Item>
          <ContextMenu.Separator />
          <ContextMenu.Item>Ferramentas do desenvolvedor</ContextMenu.Item>
        </ContextMenu.SubMenuContent>
      </ContextMenu.SubMenu>

      <ContextMenu.Separator />

      <ContextMenu.CheckboxItem
        checked={bookmarksChecked}
        onCheckedChange={setBookmarksChecked}
      >
        Mostrar favoritos
        <div>⌘+B</div>
      </ContextMenu.CheckboxItem>
      <ContextMenu.CheckboxItem
        checked={urlsChecked}
        onCheckedChange={setUrlsChecked}
      >
        Mostrar URLs completas
      </ContextMenu.CheckboxItem>

      <ContextMenu.Separator />

      <ContextMenu.Label>Pessoas</ContextMenu.Label>
      <ContextMenu.RadioGroup value={person} onValueChange={setPerson}>
        <ContextMenu.RadioItem value="pedro">
          Pedro Duarte
        </ContextMenu.RadioItem>
        <ContextMenu.RadioItem value="colm">Colm Tuite</ContextMenu.RadioItem>
      </ContextMenu.RadioGroup>
    </ContextMenu.Content>
  </ContextMenu.Root>
);
```

## Sobre

<strong>Rempi-UI Context Menu</strong> é baseado no [Radix Context Menu](https://www.radix-ui.com/primitives/docs/components/context-menu).
