# Container

<Description text="O contêiner centraliza seu conteúdo horizontalmente. É o elemento de layout mais básico." />

<SelectVersion />

<PreviewComponent name="container" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/container
```

## Uso

Importe o <strong>Container</strong>.

```jsx
import { Container } from "@rempi-ui/container";
import { Typography } from "@rempi-ui/typography";

() => (
  <Container>
    <Typography>@rempi-ui/container</Typography>
  </Container>
);
```
