## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/accordion
```

## Uso

Importe o <strong>Accordion</strong> e junte todas as partes do componente.

```jsx
import { Accordion } from "@rempi-ui/accordion";

() => (
  <Accordion.Root>
    <Accordion.Item value="item-1">
      <Accordion.Header>
        <Accordion.Trigger>É acessível?</Accordion.Trigger>
      </Accordion.Header>
      <Accordion.Content>
        Sim, ele adere ao padrão de design WAI-ARIA.
      </Accordion.Content>
    </Accordion.Item>
  </Accordion.Root>
);
```

## Sobre

Rempi-UI Accordion é baseado no [Radix Accordion](https://www.radix-ui.com/primitives/docs/components/accordion).
