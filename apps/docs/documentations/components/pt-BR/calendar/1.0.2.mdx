# Calendar

<Description text="Um componente de campo de data que permite aos usuários inserir e editar datas." />

<SelectVersion />

<PreviewComponent name="calendar" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/calendar
```

## Uso

Importe o <strong>Calendar</strong>.

```jsx
import { Calendar } from "@rempi-ui/calendar";

() => <Calendar mode="single" />;
```

## Sobre

<strong>Calendar</strong> é baseado no [React DayPicker](https://react-day-picker.js.org/).
