# Skeleton

<Description text="Use para exibir um espaço reservado enquanto o conteúdo está sendo carregado." />

<SelectVersion />

<PreviewComponent name="skeleton" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/skeleton
```

## Uso

Importe o <strong>Skeleton</strong>.

```jsx
import { Skeleton } from "@rempi-ui/skeleton";

() => (
  <Card.Root>
    <Card.Header>
      <Skeleton />
      <Skeleton />
    </Card.Header>
    <Card.Content flexDirection="column">
      <Skeleton />
      <div>
        {Array(3)
          .fill(0)
          .map((_, index) => (
            <Flex key={index} flexDirection="row" alignItems="start">
              <Skeleton />
              <Flex flexDirection="column">
                <Skeleton />
                <Skeleton />
              </Flex>
            </Flex>
          ))}
      </div>
    </Card.Content>
    <Card.Footer>
      <Skeleton />
    </Card.Footer>
  </Card.Root>
);
```
