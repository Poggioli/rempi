## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/counter
```

## Uso

Importe o <strong>Counter</strong> e junte todas as partes do componente.

```jsx
import { Counter } from "@rempi-ui/counter";

() => <Counter counter={10}>Notificações</Counter>;
```
