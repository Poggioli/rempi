## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/aspect-ratio
```

## Uso

Importe o <strong>AspectRatio</strong>.

```jsx
import { AspectRatio } from "@rempi-ui/aspect-ratio";

() => (
  <AspectRatio>
    <img
      src="https://images.unsplash.com/photo-1535025183041-0991a977e25b?w=300&dpr=2&q=80"
      alt="Fotografia de paisagem por Tobias Tullius."
    />
  </AspectRatio>
);
```

## Sobre

Rempi-UI Aspect Ratio é baseado no [Radix Aspect Ratio](https://www.radix-ui.com/primitives/docs/components/aspect-ratio).
