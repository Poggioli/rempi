## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/blockquote
```

## Uso

Importe o <strong>Blockquote</strong>.

```jsx
import { Blockquote } from "@rempi-ui/blockquote";

() => <Blockquote>Olá eu sou um Blockquote</Blockquote>;
```
