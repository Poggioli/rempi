# Tooltip

<Description text="Um pop-up que exibe informações relacionadas a um elemento quando o elemento recebe foco do teclado ou o mouse paira sobre ele." />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/tooltip
```

## Uso

Importe o <strong>Tooltip</strong> e junte todas as partes do componente.

```jsx
import { IconButton } from "@rempi-ui/button";
import { Tooltip } from "@rempi-ui/tooltip";
import { HelpCircle } from "lucide-react";

() => (
  <html>
    {...}
    <Tooltip.Provider>
      <Tooltip.Root>
        <Tooltip.Trigger>
          <IconButton
            aria-label="Configurações"
            variant="text"
            shape="rounded"
            color="primary"
          >
            <HelpCircle size={16} />
          </IconButton>
        </Tooltip.Trigger>
        <Tooltip.Content>
          Adicionar biblioteca
          <Tooltip.Arrow />
        </Tooltip.Content>
      </Tooltip.Root>
    </Tooltip.Provider>
  </html>
);
```

## Sobre

<strong>Rempi-UI Tooltip</strong> é baseado no [Radix Tooltip](https://www.radix-ui.com/primitives/docs/components/tooltip).

O botão que ativa ou desativa a dica de ferramenta. Por padrão, o `Tooltip.Content` se posicionará em relação ao gatilho.

### Content

É uma mistura de [Radix Tooltip Content](https://www.radix-ui.com/primitives/docs/components/tooltip#content) e [Radix Tooltip Portal](https://www.radix-ui.com/primitives/docs/components/tooltip#portal).<br/><br/>

Quando usado, move a parte de conteúdo para o elemento `body`. O componente que aparece quando o tooltip está aberto.
