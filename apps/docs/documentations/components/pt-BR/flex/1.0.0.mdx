# Flex

<Description text="Componente para criar layouts flexíveis." />

<SelectVersion />

<PreviewComponent name="flex" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/flex
```

## Uso

Importe o <strong>Flex</strong>.

```jsx
import { Flex } from "@rempi-ui/flex";
import { Typography } from "@rempi-ui/typography";

() => (
  <Flex alignItems="center" justifyContents="center">
    <Typography>Flex</Typography>
  </Flex>
);
```
