## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/dialog
```

## Uso

Importe o <strong>Dialog</strong> e junte todas as partes do componente.

```jsx
import { Button } from "@rempi-ui/button";
import { Dialog } from "@rempi-ui/dialog";
import { Flex } from "@rempi-ui/flex";
import { Input } from "@rempi-ui/input";

() => (
  <Dialog.Root>
    <Dialog.Trigger>
      <Button>Editar perfil</Button>
    </Dialog.Trigger>
    <Dialog.Portal>
      <Dialog.Overlay blur />
      <Dialog.Content>
        <Dialog.CloseCross aria-label="Fechar" />
        <Dialog.Title>Editar perfil</Dialog.Title>
        <Dialog.Description>
          Faça alterações em seu perfil aqui. Clique em salvar quando terminar.
        </Dialog.Description>
        <Flex
          flexDirection="column"
          flexWrap="wrap"
          justifyContent="space-between"
        >
          <Input.Label alignItems="flex-start" flexDirection="column">
            Nome
            <Input.Root placeholder="Digite seu nome" />
          </Input.Label>
          <Input.Label alignItems="flex-start" flexDirection="column">
            Nome de usuário
            <Input.Root placeholder="Digite seu nome de usuário" />
          </Input.Label>
        </Flex>
        <Dialog.Footer>
          <Dialog.Close>
            <Button color="primary" variant="text">
              Cancelar
            </Button>
          </Dialog.Close>
          <Dialog.Close>
            <Button color="success">Salvar</Button>
          </Dialog.Close>
        </Dialog.Footer>
      </Dialog.Content>
    </Dialog.Portal>
  </Dialog.Root>
);
```

## Sobre

<strong>Rempi-UI Dialog</strong> é baseado no [Radix Dialog](https://www.radix-ui.com/primitives/docs/components/dialog).
