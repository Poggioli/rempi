# Hover Card

<Description text="Pré-visualização de conteúdo disponível por trás de um link." />

<SelectVersion />

<PreviewComponent name="hover-card" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/hover-card
```

## Uso

Importe o <strong>HoverCard</strong> e junte todas as partes do componente.

```jsx
import { HoverCard } from "@rempi-ui/hover-card";
import { Typography } from "@rempi-ui/typography";

() => (
  <HoverCard.Root>
    <HoverCard.Trigger>
      <a
        href="https://twitter.com/_Poggioli"
        target="_blank"
        rel="noreferrer noopener"
      >
        <img
          src="https://pbs.twimg.com/profile_images/1639359724691808262/UErOMnL2_400x400.jpg"
          alt="foto de perfil do Twitter de @_Poggioli"
        />
      </a>
    </HoverCard.Trigger>
    <HoverCard.Content>
      <Flex flexDirection="column">
        <img
          src="https://pbs.twimg.com/profile_images/1639359724691808262/UErOMnL2_400x400.jpg"
          alt="foto de perfil do Twitter de @_Poggioli"
        />
        <Flex flexDirection="column">
          <Flex flexDirection="column">
            <Typography
              as="h6"
              color="high-contrast"
              fontWeight="bold"
              fontSize="md"
              lineHeight={2}
            >
              Poggioli
            </Typography>
            <Typography color="low-contrast" fontSize="sm" lineHeight={2}>
              @_Poggioli
            </Typography>
          </Flex>
          <Typography color="high-contrast" fontSize="md" lineHeight={1}>
            Dev Frontend; A11Y web enthusiast; I love birds, especially my
            cockatiel; I love my angry girlfriend;
          </Typography>
          <Flex alignItems="center">
            <Flex alignItems="center">
              <Typography
                color="high-contrast"
                fontWeight="bold"
                fontSize="md"
                lineHeight={2}
              >
                282
              </Typography>
              <Typography color="low-contrast" fontSize="sm" lineHeight={2}>
                Seguindo
              </Typography>
            </Flex>
            <Flex alignItems="center">
              <Typography
                color="high-contrast"
                fontWeight="bold"

                lineHeight={2}
              >
                454
              </Typography>
              <Typography color="low-contrast" fontSize="sm" lineHeight={2}>
                Seguidores
              </Typography>
            </Flex>
          </Flex>
        </Flex>
      </Flex>
      <HoverCard.Arrow />
    </HoverCard.Content>
  </HoverCard.Root>
);
```

## Sobre

Rempi-UI Hover Card é baseado no [Radix Hover Card](https://www.radix-ui.com/primitives/docs/components/hover-card).

### Content

É uma mistura de [Radix Hover Card Content](https://www.radix-ui.com/primitives/docs/components/hover-card#content) e [Radix Hover Card Portal](https://www.radix-ui.com/primitives/docs/components/hover-card#portal).<br/><br/>
Quando usado, move a parte de conteúdo para o elemento `body`. O componente que aparece quando o hover card está aberto.

Um elemento de seta opcional para renderizar junto com o hover card. Isso pode ser usado para ajudar a vincular visualmente o disparador ao `HoverCard.Content`. Deve ser renderizado dentro de `HoverCard.Content`.
