## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/switch
```

## Uso

Importe o <strong>Switch</strong> e junte todas as partes do componente.

```jsx
import { Flex } from "@rempi-ui/Flex";
import { Switch } from "@rempi-ui/switch";
import { Typography } from "@rempi-ui/typography";
import { BellIcon } from "lucide-react";

() => (
  <Switch.Label alignItems="center">
    <BellIcon size={16} />
    <Flex flexDirection="column">
      <Typography
        fontSize="sm"
        fontWeight="semi-bold"
        color="high-contrast"
        lineHeight={1}
      >
        Notificações via push
      </Typography>
      <Typography
        fontSize="sm"
        fontWeight="regular"
        color="low-contrast"
        lineHeight={1}
      >
        Envie notificações para o dispositivo.
      </Typography>
    </Flex>
    <Switch.Thumb />
  </Switch.Label>
);
```

## Sobre

Rempi-UI Switch é baseado no [Radix Switch](https://www.radix-ui.com/primitives/docs/components/switch).

## API reference

### Thumb

É uma mistura de [Radix Switch Root](https://www.radix-ui.com/primitives/docs/components/switch#root) e [Radix Switch Thumb](https://www.radix-ui.com/primitives/docs/components/switch#thumb).<br/><br/>
Contém todas as partes do switch. Um `input` também será renderizado quando usado dentro de um `form` para garantir que os eventos sejam propagados corretamente. O marcador que é usado para indicar visualmente se o switch está ligado ou desligado.
