## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/navigation-menu
```

## Uso

Importe o <strong>NavigationMenu</strong> e junte todas as partes do componente.

```jsx
import { NavigationMenu } from "@rempi-ui/navigation-menu";

() => (
  <NavigationMenu.Root>
    <NavigationMenu.Menu>
      <NavigationMenu.Trigger>Arquivo</NavigationMenu.Trigger>
      <NavigationMenu.Content align="start">
        <NavigationMenu.Item>Nova aba</NavigationMenu.Item>
        <NavigationMenu.Item>Nova janela</NavigationMenu.Item>
        <NavigationMenu.Item disabled>Nova janela anônima</NavigationMenu.Item>
        <NavigationMenu.Separator />
        <NavigationMenu.SubMenu>
          <NavigationMenu.SubMenuTrigger>
            Compartilhar
          </NavigationMenu.SubMenuTrigger>
          <NavigationMenu.SubMenuContent alignOffset={-5}>
            <NavigationMenu.Item>Email Link</NavigationMenu.Item>
            <NavigationMenu.Item>Mensagem</NavigationMenu.Item>
            <NavigationMenu.Item>Notas</NavigationMenu.Item>
          </NavigationMenu.SubMenuContent>
        </NavigationMenu.SubMenu>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Imprimir...</NavigationMenu.Item>
      </NavigationMenu.Content>
    </NavigationMenu.Menu>

    <NavigationMenu.Menu>
      <NavigationMenu.Trigger>Editar</NavigationMenu.Trigger>
      <NavigationMenu.Content align="start">
        <NavigationMenu.Item>Desfazer</NavigationMenu.Item>
        <NavigationMenu.Item>Refazer</NavigationMenu.Item>
        <NavigationMenu.Separator />
        <NavigationMenu.SubMenu>
          <NavigationMenu.SubMenuTrigger>Buscar</NavigationMenu.SubMenuTrigger>

          <NavigationMenu.SubMenuContent alignOffset={-5}>
            <NavigationMenu.Item>Buscar na internet...</NavigationMenu.Item>
            <NavigationMenu.Separator />
            <NavigationMenu.Item>Buscar</NavigationMenu.Item>
            <NavigationMenu.Item>Buscar Próximo</NavigationMenu.Item>
            <NavigationMenu.Item>Buscar Anterior</NavigationMenu.Item>
          </NavigationMenu.SubMenuContent>
        </NavigationMenu.SubMenu>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Cortar</NavigationMenu.Item>
        <NavigationMenu.Item>Copiar</NavigationMenu.Item>
        <NavigationMenu.Item>Colar</NavigationMenu.Item>
      </NavigationMenu.Content>
    </NavigationMenu.Menu>

    <NavigationMenu.Menu>
      <NavigationMenu.Trigger>Visualizar</NavigationMenu.Trigger>
      <NavigationMenu.Content align="start" alignOffset={-14}>
        <NavigationMenu.CheckboxItem value="always-show-bookmarks-bar">
          Sempre mostrar a barra de favoritos
        </NavigationMenu.CheckboxItem>
        <NavigationMenu.CheckboxItem value="always-show-full-urls">
          Sempre mostrar as URL's completas
        </NavigationMenu.CheckboxItem>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Recarregar</NavigationMenu.Item>
        <NavigationMenu.Item disabled>
          Recarregamento forçado
        </NavigationMenu.Item>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Alternar para tela cheia</NavigationMenu.Item>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Esconder menu lateral</NavigationMenu.Item>
      </NavigationMenu.Content>
    </NavigationMenu.Menu>

    <NavigationMenu.Menu>
      <NavigationMenu.Trigger disabled>Empresas</NavigationMenu.Trigger>
    </NavigationMenu.Menu>
  </NavigationMenu.Root>
);
```

## Sobre

<strong>Rempi-UI Navigation Menu</strong> é baseado no [Radix Navigation Menu](https://www.radix-ui.com/primitives/docs/components/navigation-menu).

## API reference

### Root

É uma mistura de [Radix Navigation Menu Root](https://www.radix-ui.com/primitives/docs/components/navigation-menu#root), [Radix Navigation Menu List](https://www.radix-ui.com/primitives/docs/components/navigation-menu#list) e [Radix Navigation Menu Viewport](https://www.radix-ui.com/primitives/docs/components/navigation-menu#viewport).<br/><br/>

Contém todas as partes do navigation menu. Contém os itens de menu de navegação. Um elemento de viewport opcional que é usado para renderizar conteúdo ativo fora da lista.
