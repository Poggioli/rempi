## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/progress
```

## Uso

Importe o <strong>Progress</strong>.

```jsx
import { Progress } from "@rempi-ui/progress";

() => <Progress value={75} />;
```

## Sobre

<strong>Rempi-UI Progress</strong> é baseado no [Radix Progress](https://www.radix-ui.com/primitives/docs/components/progress).

## API reference

### Progress

É uma mistura de [Radix Progress Root](https://www.radix-ui.com/primitives/docs/components/progress#root) e [Radix Progress Indicator](https://www.radix-ui.com/primitives/docs/components/progress#indicator).<br/><br/>

Contém todas as partes do progresso. Usado para mostrar o progresso visualmente. Também torna o progresso acessível às tecnologias assistivas.
