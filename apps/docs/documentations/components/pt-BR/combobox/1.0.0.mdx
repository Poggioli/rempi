# Combobox

<Description text="Combobox com uma lista de sugestões." />

<SelectVersion />

<PreviewComponent name="combobox" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/combobox
```

## Uso

Importe o <strong>Combobox</strong> e junte todas as partes do componente.

```jsx
import { Combobox } from "@rempi-ui/combobox";

() => (
  <Combobox.Root>
    <Combobox.Trigger>
      <Combobox.Value placeholder="Selecione framework..." />
    </Combobox.Trigger>
    <Combobox.Content>
      <Combobox.Input placeholder="Busque framework..." />
      <Combobox.Empty>
        Framework não encontrado.
      </Combobox.Empty>
      <Combobox.Label>Backend</Combobox.Label>
      <Combobox.Item value="spring-boot">SpringBoot</Combobox.Item>
      <Combobox.Item value="django" disabled>
        Django
      </Combobox.Item>
      <Combobox.Item value="laravel">Laravel</Combobox.Item>
      <Combobox.Item value="express-js">Express.js</Combobox.Item>
      <Combobox.Item value="ruby-on-rails">Ruby on Rails</Combobox.Item>
      <Combobox.Separator />
      <Combobox.Label>Frontend</Combobox.Label>
      <Combobox.Item value="next-js">Next.js</Combobox.Item>
      <Combobox.Item value="svelte-kit">SvelteKit</Combobox.Item>
      <Combobox.Item value="remix">Remix</Combobox.Item>
      <Combobox.Item value="astro">Astro</Combobox.Item>
    </Combobox.Content>
  </Combobox.Root>
);
```

### Trigger

O elemento que ativa o popover. Por padrão, o <code>Popover.Content</code> se posicionará em relação ao gatilho.

<MoreInfo
  componentName="Rempi-UI Popover Trigger"
  componentLink="/docs/components/popover"
/>

### Value

A parte que reflete o valor selecionado. Por padrão, o texto do item selecionado será renderizado. Se você precisar de mais controle,
pode controlar o select e passar seus próprios filhos. Não deve ser estilizado para garantir um posicionamento correto. Uma propriedade
de espaço reservado opcional também está disponível quando o select não tem valor.

### Label

Usado para renderizar o rótulo de um grupo. Não será focável usando as teclas de seta.
