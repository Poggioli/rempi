## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/select
```

## Uso

Importe o <strong>Select</strong> e junte todas as partes do componente.

```jsx
import { Select } from "@rempi-ui/select";

() => (
  <Select.Root>
    <Select.Trigger aria-label="Comida">
      <Select.Value placeholder="Selecione a comida..." />
    </Select.Trigger>
    <Select.Content>
      <Select.Viewport>
        <Select.Group>
          <Select.Label>Frutas</Select.Label>
          <Select.Item value="apple">Maçã</Select.Item>
          <Select.Item disabled value="banana">
            Banana
          </Select.Item>
          <Select.Item value="blueberry">Mirtilo</Select.Item>
          <Select.Item value="grapes">Uva</Select.Item>
          <Select.Item value="pineapple">Abacaxi</Select.Item>
        </Select.Group>

        <Select.Separator />

        <Select.Group>
          <Select.Label>Vegetais</Select.Label>
          <Select.Item value="aubergine">Berinjela</Select.Item>
          <Select.Item value="broccoli">Brócolis</Select.Item>
          <Select.Item value="carrot" disabled>
            Cenoura
          </Select.Item>
          <Select.Item value="courgette">Abobrinha</Select.Item>
          <Select.Item value="leek">Alho-poró</Select.Item>
        </Select.Group>

        <Select.Separator />

        <Select.Group>
          <Select.Label>Carnes</Select.Label>
          <Select.Item value="beef">Carne bovina</Select.Item>
          <Select.Item value="chicken">Frango</Select.Item>
          <Select.Item value="lamb">Ovelha</Select.Item>
          <Select.Item value="pork">Carne suína</Select.Item>
        </Select.Group>
      </Select.Viewport>
    </Select.Content>
  </Select.Root>
);
```

## Sobre

Rempi-UI Select é baseado no [Radix Select](https://www.radix-ui.com/primitives/docs/components/select).

### Value

A parte que reflete o valor selecionado. Por padrão, o texto do item selecionado será renderizado.
Se você precisar de mais controle, pode controlar o select e passar seu próprio `children`.
Ele não deve ser estilizado para garantir o posicionamento correto. Uma propriedade opcional de `placeholder`
também está disponível quando o select não tem valor.

### Item

É uma mistura de [Radix-UI Select Item](https://www.radix-ui.com/primitives/docs/components/select#item), [Radix-UI Select Item Text](https://www.radix-ui.com/primitives/docs/components/select#itemtext) e [Radix-UI Select Item Indicator](https://www.radix-ui.com/primitives/docs/components/select#itemindicator).<br/><br/>
O componente que contém os itens do select. A parte textual do item. Deve conter apenas o texto que você deseja ver no gatilho quando esse item for selecionado. Não deve ser estilizado para garantir o posicionamento correto. É renderizado quando o item é selecionado. Você pode estilizar este elemento diretamente ou usá-lo como um invólucro para inserir um ícone, ou ambos.
