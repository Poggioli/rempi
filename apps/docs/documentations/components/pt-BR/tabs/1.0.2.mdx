## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/tabs
```

## Uso

Importe o <strong>Tabs</strong> e junte todas as partes do componente.

```jsx
import { Avatar } from "@rempi-ui/avatar";
import { Flex } from "@rempi-ui/flex";
import { Tabs } from "@rempi-ui/tabs";
import { Typography } from "@rempi-ui/typography";

() => (
  <Tabs.Root defaultValue="photos">
    <Tabs.List>
      <Tabs.Trigger value="photos">Fotos</Tabs.Trigger>
      <Tabs.Trigger value="description">Descrição</Tabs.Trigger>
      <Tabs.Trigger value="personal-settings" disabled>
        Configurações pessoais
      </Tabs.Trigger>
    </Tabs.List>
    <Tabs.Content value="photos">
      <Flex

      >
```

## Sobre

Rempi-UI Tabs é baseado no [Radix Tabs](https://www.radix-ui.com/primitives/docs/components/tabs).
