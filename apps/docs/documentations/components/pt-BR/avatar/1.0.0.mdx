## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/avatar
```

## Uso

Importe o <strong>Avatar</strong> e junte todas as partes do componente.

```jsx
import { Avatar } from "@rempi-ui/avatar";

() => (
  <Avatar.Root size="standard">
    <Avatar.Image
      src="https://images.unsplash.com/photo-1492633423870-43d1cd2775eb?&w=128&h=128&dpr=2&q=80"
      alt="Colm Tuite"
    />
    <Avatar.Fallback delayMs={600}>CT</Avatar.Fallback>
  </Avatar.Root>
);
```

## Sobre

Rempi-UI Avatar é baseado no [Radix Avatar](https://www.radix-ui.com/primitives/docs/components/avatar).

### Fallback

Um elemento que é renderizado quando a imagem não foi carregada. Isso significa durante o carregamento da imagem ou se houve um erro. Se você perceber um flash durante o carregamento, pode fornecer uma propriedade `delayMs` para atrasar sua renderização, de modo que ela seja renderizada apenas para aqueles com conexões mais lentas. Para um controle mais preciso, use o evento `onLoadingStatusChange` em `Avatar.Image`.
