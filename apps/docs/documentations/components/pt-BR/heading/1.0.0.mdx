# Heading

<Description text="Um elemento de cabeçalho semântico." />

<SelectVersion />

<PreviewComponent name="heading" />

## Instalação

Instale o componente a partir da linha de comando.

```bash
yarn add @rempi-ui/heading
```

## Uso

Importe o <strong>Heading</strong>.

```jsx
import { Heading } from "@rempi-ui/heading";
import { Typography } from "@rempi-ui/typography";

() => (
  <Heading alignItems="center" justifyContents="center">
    <Typography>Olá eu sou um Heading</Typography>
  </Heading>
);
```
