## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/toggle
```

## Usage

Import the <strong>Toggle</strong>.

```jsx
import { Flex } from "@rempi-ui/Flex";
import { Toggle } from "@rempi-ui/toggle";
import { Italic } from "lucide-react";

() => (
  <Flex alignItems="center" justifyContent="center">
    <Toggle>
      <Italic size={16} />
    </Toggle>

    <Toggle defaultPressed>
      <Italic size={16} />
    </Toggle>

    <Toggle disabled>
      <Italic size={16} />
    </Toggle>

    <Toggle disabled defaultPressed>
      <Italic size={16} />
    </Toggle>
  </Flex>
);
```

## About

<strong>Rempi-UI Toggle</strong> component is built on top of [Radix Toggle](https://www.radix-ui.com/primitives/docs/components/toggle).
