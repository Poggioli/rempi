## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/progress
```

## Usage

Import the <strong>Progress</strong>.

```jsx
import { Progress } from "@rempi-ui/progress";

() => <Progress value={75} />;
```

## About

<strong>Rempi-UI Progress</strong> component is built on top of [Radix Progress](https://www.radix-ui.com/primitives/docs/components/progress).

## API reference

### Progress

It's a mix of [Radix Progress Root](https://www.radix-ui.com/primitives/docs/components/progress#root) and [Radix Progress Indicator](https://www.radix-ui.com/primitives/docs/components/progress#indicator).<br/><br/>

Contains all of the progress parts. Used to show the progress visually. It also makes progress accessible to assistive technologies.
