# Counter

<Description text="Generates a small counter in the corner of its child components" />

<SelectVersion />

<PreviewComponent name="counter" />

## Installation

Install the component from the command line.

```bash
yarn add @rempi-ui/counter
```

## Usage

Import the <strong>Counter</strong> and piece all composition components together.

```jsx
import { Counter } from "@rempi-ui/counter";

() => <Counter counter={10}>Notifications</Counter>;
```
