# Code

<Description text="Marks text to signify a short fragment of computer code." />

<SelectVersion />

<PreviewComponent name="code" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/code
```

## Usage

Import the <strong>Code</strong>.

```jsx
import { Code } from "@rempi-ui/code";

() => <Code>console.log('Hello I'm a Code')</Code>;
```
