## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/switch
```

## Usage

Import the <strong>Switch</strong> and piece all composition components together.

```jsx
import { Flex } from "@rempi-ui/flex";
import { Switch } from "@rempi-ui/switch";
import { Typography } from "@rempi-ui/typography";
import { BellIcon } from "lucide-react";

() => (
  <Switch.Label alignItems="center">
    <BellIcon size={16} />
    <Flex flexDirection="column">
      <Typography
        fontSize="sm"
        fontWeight="semi-bold"
        color="high-contrast"
        lineHeight={1}
      >
        Push Notifications
      </Typography>
      <Typography
        fontSize="sm"
        fontWeight="regular"
        color="low-contrast"
        lineHeight={1}
      >
        Send notifications to device.
      </Typography>
    </Flex>
    <Switch.Thumb />
  </Switch.Label>
);
```

## About

Rempi-UI Switch component is built on top of [Radix Switch](https://www.radix-ui.com/primitives/docs/components/switch).

## API reference

### Thumb

It's a mix of [Radix Switch Root](https://www.radix-ui.com/primitives/docs/components/switch#root) and [Radix Switch Thumb](https://www.radix-ui.com/primitives/docs/components/switch#thumb).<br/><br/>
Contains all the parts of a switch. An `input` will also render when used within a `form` to ensure events propagate correctly. The thumb that is used to visually indicate whether the switch is on or off.
