# Label

<Description text="Renders an accessible label associated with controls." />

<SelectVersion />

<PreviewComponent name="label" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/label
```

## Usage

Import the <strong>Label</strong>.

```jsx
import { Label } from "@rempi-ui/label";

() => <Label>Hi I'm Label</Label>;
```

## About

Rempi-UI Label component is built on top of [Radix Label](https://www.radix-ui.com/primitives/docs/components/label).

## API reference

### Label

It's a mix of [Radix Label](https://www.radix-ui.com/primitives/docs/components/label) and [Rempi-UI Flex](docs/flex). <br /><br />
Contains the content for the label. Has all props of [Rempi-UI Flex](docs/flex).
