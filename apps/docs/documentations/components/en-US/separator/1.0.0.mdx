## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/separator
```

## Usage

Import the <strong>Separator</strong>.

```jsx
import { Separator } from "@rempi-ui/separator";

() => (
  <Flex flexDirection="column">
    <Heading>Between us there</Heading>
    <Separator />
    <Heading>Is a separator</Heading>
  </Flex>
);
```

## About

Rempi-UI Separator component is built on top of [Radix Separator](https://www.radix-ui.com/primitives/docs/components/separator).
