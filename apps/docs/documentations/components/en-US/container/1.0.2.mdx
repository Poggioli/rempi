# Container

<Description text="The container centers your content horizontally. It's the most basic layout element." />

<SelectVersion />

<PreviewComponent name="container" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/container
```

## Usage

Import the <strong>Container</strong>.

```jsx
import { Container } from "@rempi-ui/container";
import { Typography } from "@rempi-ui/typography";

() => (
  <Container>
    <Typography>@rempi-ui/container</Typography>
  </Container>
);
```
