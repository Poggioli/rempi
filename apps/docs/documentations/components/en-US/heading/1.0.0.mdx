# Heading

<Description text="A semantic heading element." />

<SelectVersion />

<PreviewComponent name="heading" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/heading
```

## Usage

Import the <strong>Heading</strong>.

```jsx
import { Heading } from "@rempi-ui/heading";
import { Typography } from "@rempi-ui/typography";

() => (
  <Heading alignItems="center" justifyContents="center">
    <Typography>Heading</Typography>
  </Heading>
);
```
