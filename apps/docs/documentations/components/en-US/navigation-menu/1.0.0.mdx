## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/navigation-menu
```

## Usage

Import the <strong>NavigationMenu</strong> and piece all composition components together.

```jsx
import { NavigationMenu } from "@rempi-ui/navigation-menu";

() => (
  <NavigationMenu.Root>
    <NavigationMenu.Menu>
      <NavigationMenu.Trigger>File</NavigationMenu.Trigger>
      <NavigationMenu.Content align="start">
        <NavigationMenu.Item>New Tab</NavigationMenu.Item>
        <NavigationMenu.Item>New Window</NavigationMenu.Item>
        <NavigationMenu.Item disabled>New Incognito Window</NavigationMenu.Item>
        <NavigationMenu.Separator />
        <NavigationMenu.SubMenu>
          <NavigationMenu.SubMenuTrigger>Share</NavigationMenu.SubMenuTrigger>
          <NavigationMenu.SubMenuContent alignOffset={-5}>
            <NavigationMenu.Item>Email Link</NavigationMenu.Item>
            <NavigationMenu.Item>Messages</NavigationMenu.Item>
            <NavigationMenu.Item>Notes</NavigationMenu.Item>
          </NavigationMenu.SubMenuContent>
        </NavigationMenu.SubMenu>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Print…</NavigationMenu.Item>
      </NavigationMenu.Content>
    </NavigationMenu.Menu>

    <NavigationMenu.Menu>
      <NavigationMenu.Trigger>Edit</NavigationMenu.Trigger>
      <NavigationMenu.Content align="start">
        <NavigationMenu.Item>Undo</NavigationMenu.Item>
        <NavigationMenu.Item>Redo</NavigationMenu.Item>
        <NavigationMenu.Separator />
        <NavigationMenu.SubMenu>
          <NavigationMenu.SubMenuTrigger>Find</NavigationMenu.SubMenuTrigger>

          <NavigationMenu.SubMenuContent alignOffset={-5}>
            <NavigationMenu.Item>Search the web…</NavigationMenu.Item>
            <NavigationMenu.Separator />
            <NavigationMenu.Item>Find…</NavigationMenu.Item>
            <NavigationMenu.Item>Find Next</NavigationMenu.Item>
            <NavigationMenu.Item>Find Previous</NavigationMenu.Item>
          </NavigationMenu.SubMenuContent>
        </NavigationMenu.SubMenu>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Cut</NavigationMenu.Item>
        <NavigationMenu.Item>Copy</NavigationMenu.Item>
        <NavigationMenu.Item>Paste</NavigationMenu.Item>
      </NavigationMenu.Content>
    </NavigationMenu.Menu>

    <NavigationMenu.Menu>
      <NavigationMenu.Trigger>View</NavigationMenu.Trigger>
      <NavigationMenu.Content align="start" alignOffset={-14}>
        <NavigationMenu.CheckboxItem value="always-show-bookmarks-bar">
          Always Show Bookmarks Bar
        </NavigationMenu.CheckboxItem>
        <NavigationMenu.CheckboxItem value="always-show-full-urls">
          Always Show Full URLs
        </NavigationMenu.CheckboxItem>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Reload</NavigationMenu.Item>
        <NavigationMenu.Item disabled>Force Reload</NavigationMenu.Item>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Toggle Fullscreen</NavigationMenu.Item>
        <NavigationMenu.Separator />
        <NavigationMenu.Item>Hide Sidebar</NavigationMenu.Item>
      </NavigationMenu.Content>
    </NavigationMenu.Menu>

    <NavigationMenu.Menu>
      <NavigationMenu.Trigger disabled>Companies</NavigationMenu.Trigger>
    </NavigationMenu.Menu>
  </NavigationMenu.Root>
);
```

## About

<strong>Rempi-UI Navigation Menu</strong> component is built on top of [Radix Navigation
Menu](https://www.radix-ui.com/primitives/docs/components/navigation-menu).

## API reference

### Root

It's a mix of [Radix Navigation Menu Root](https://www.radix-ui.com/primitives/docs/components/navigation-menu#root), [Radix Navigation Menu List](https://www.radix-ui.com/primitives/docs/components/navigation-menu#list) and [Radix Navigation Menu Viewport](https://www.radix-ui.com/primitives/docs/components/navigation-menu#viewport).<br/><br/>

Contains all the parts of a navigation menu. Contains the top level menu items. An optional viewport element that is used to render active content outside of the list.
