# Customização

<Description text="Rempi-UI was designed so that you can customize tokens and make the library look like your product." />

<Steps>

<Step>Create the theme</Step>

To create a custom theme, you must import the <strong>createTheme</strong> function
from the <strong>@rempi-ui/core</strong> library.

<Step>Using the createTheme function</Step>

The first parameter that the <strong>createTheme</strong> function receives is the name
of the theme you are creating, in this example we will create a clear theme.

```jsx
import { createTheme } from "@rempi-ui/core";

const lightTheme = createTheme("light-theme", {});
```

This is the current structure of the tokens, the second parameter of the function reflects the
type <strong>RempiConfigCreateTheme</strong>. Therefore, you are not obliged
to pass all tokens to customize a theme, but keep in mind that the <strong>Rempi-UI</strong> library
will default to the values not informed by the session tokens.
[Tokens](/docs/overview/tokens).

This is why it is important to provide all tokens from the same group, e.g.
of the primary colors, so that the library functions correctly.

The colors used were taken from the library [Radix Colors](https://www.radix-ui.com/colors).
