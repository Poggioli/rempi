# Customização

<Description text="Rempi-UI foi pensada para que você possa customizar tokens e deixar a biblioteca com a cara do seu produto." />

<Steps>

<Step>Criar o tema</Step>

Para Criar um tema customizado, você deve importar a função de <strong>createTheme</strong>
da biblioteca <strong>@rempi-ui/core</strong>.

<Step>Utilizando a função createTheme</Step>

O Primeiro parâmetro que a função <strong>createTheme</strong> recebe é o nome
do tema que você está criando, neste exemplo vamos criar um tema claro.

```jsx
import { createTheme } from "@rempi-ui/core";

const lightTheme = createTheme("light-theme", {});
```

Essa é a estrutura atual dos tokens, o segundo parâmetro da função reflete o
tipo <strong>RempiConfigCreateTheme</strong>. Sendo assim você não é obrigado
a passar todos os tokens para customizar um tema, mas tenha em mente que a biblioteca <strong>Rempi-UI</strong>
irá usar como padrão para os valores não informados os tokens da sessão de
[Tokens](/docs/overview/tokens).

Por isso é importante fornecer todos os tokens de um mesmo grupo, por exemplo
das cores primárias, para que haja um funcionamento correto da biblioteca.

As cores utilizadas foram retiradas da biblioteca [Radix Colors](https://www.radix-ui.com/colors).
