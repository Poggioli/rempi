# Calendar

<Description text="A date field component that allows users to enter and edit date." />

## Example

<PreviewComponent name="calendar" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/calendar
```

## Usage

Import the <strong>Calendar</strong>.

```jsx
import { Calendar } from "@rempi-ui/calendar";

() => <Calendar mode="single" />;
```

## About

The <strong>Calendar</strong> component is built on top of [React DayPicker](https://react-day-picker.js.org/).
