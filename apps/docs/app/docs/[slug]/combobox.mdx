# Combobox

<Description>Autocomplete input and command palette with a list of suggestions.</Description>

## Example

<PreviewComponent name="combobox" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/combobox
```

## Usage

Import the <strong>Combobox</strong> and piece all composition components together.

```jsx
import { Combobox } from "@rempi-ui/combobox";

() => (
  <Combobox.Root>
    <Combobox.Trigger>
      <Combobox.Value placeholder="Select framework..." />
    </Combobox.Trigger>
    <Combobox.Content>
      <Combobox.Input placeholder="Search framework..." />
      <Combobox.Empty style={{ maxWidth: 180 }}>
        No framework found.
      </Combobox.Empty>
      <Combobox.Label>Backend</Combobox.Label>
      <Combobox.Item value="spring-boot">SpringBoot</Combobox.Item>
      <Combobox.Item value="django" disabled>
        Django
      </Combobox.Item>
      <Combobox.Item value="laravel">Laravel</Combobox.Item>
      <Combobox.Item value="express-js">Express.js</Combobox.Item>
      <Combobox.Item value="ruby-on-rails">Ruby on Rails</Combobox.Item>
      <Combobox.Separator />
      <Combobox.Label>Frontend</Combobox.Label>
      <Combobox.Item value="next-js">Next.js</Combobox.Item>
      <Combobox.Item value="svelte-kit">SvelteKit</Combobox.Item>
      <Combobox.Item value="remix">Remix</Combobox.Item>
      <Combobox.Item value="astro">Astro</Combobox.Item>
    </Combobox.Content>
  </Combobox.Root>
);
```

### Trigger

For more information access [Rempi-UI Popover Trigger](/docs/popover) documentation.

### Value

The part that reflects the selected value. By default the selected item's text will be rendered. if you require more control, you can instead control the select and pass your own children. It should not be styled to ensure correct positioning. An optional placeholder prop is also available for when the select has no value.

### Separator

Used to visually separate items in the select.<br />
For more information access [Rempi-UI Popover Separator](/docs/separator) documentation.

### Input

Used to user insert the value to be searched.<br />
For more information access [Rempi-UI Popover Input](/docs/input) documentation.

### Empty

Used to indicates when searched value not found any correspondence.

### Label

Used to render the label of a group. It won't be focusable using arrow keys.
