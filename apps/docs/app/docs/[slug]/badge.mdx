# Badge

<Description>Are compact elements that represent an input, attribute, or action.</Description>

## Example

<PreviewComponent name="badge" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/badge
```

## Usage

Import the <strong>Badge</strong>.

```jsx
import { Badge } from "@rempi-ui/badge";

() => <Badge>Hello I'm a Badge</Badge>;
```
