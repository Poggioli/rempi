# Badge

Displays a badge or a component that looks like a badge.

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/badge
```

Import the <strong>Badge</strong>.

```jsx

() => <Badge>Hello I'm a Badge</Badge>;
```
