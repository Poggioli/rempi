# Card

<Description>Displays a card with header, content, and footer.</Description>

## Example

<PreviewComponent name="card" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/card
```

## Usage

Import the <strong>Card</strong> and piece all composition components together.

```jsx
import { Button } from "@rempi-ui/button";
import { Card } from "@rempi-ui/card";
import { Flex } from "@rempi-ui/flex";
import { Switch } from "@rempi-ui/switch";
import { Typography } from "@rempi-ui/typography";
import { BellIcon, Check } from "lucide-react";

() => (
  <Card.Root>
    <Card.Header>
      <Card.Title>Notifications</Card.Title>
      <Card.Description>You have 3 unread messages.</Card.Description>
    </Card.Header>
    <Card.Content flexDirection="column">
      <Flex alignItems="center">
        <BellIcon size={16} />
        <Flex flexDirection="column">
          <Typography
            fontSize="sm"
            fontWeight="semi-bold"
            color="high-contrast"
            lineHeight={1}
          >
            Push Notifications
          </Typography>
          <Typography
            fontSize="sm"
            fontWeight="regular"
            color="low-contrast"
            lineHeight={1}
          >
            Send notifications to device.
          </Typography>
        </Flex>
        <Switch.Thumb />
      </Flex>
      <Flex flexDirection="row" alignItems="start">
        <Typography />
        <Flex flexDirection="column">
          <Typography
            fontSize="sm"
            fontWeight="semi-bold"
            color="high-contrast"
            lineHeight={1}
          >
            Your call has been confirmed.
          </Typography>
          <Typography
            fontSize="sm"
            fontWeight="regular"
            color="low-contrast"
            lineHeight={1}
          >
            1 hour ago
          </Typography>
        </Flex>
      </Flex>
    </Card.Content>
    <Card.Footer>
      <Button>
        <Check size={16} />
        Mark all as read
      </Button>
    </Card.Footer>
  </Card.Root>
);
```
