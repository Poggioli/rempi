# Skeleton

<Description text="Use to show a placeholder while content is loading." />

## Example

<PreviewComponent name="skeleton" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/skeleton
```

## Usage

Import the <strong>Skeleton</strong>.

```jsx
import { Skeleton } from "@rempi-ui/skeleton";

() => (
  <Card.Root>
    <Card.Header>
      <Skeleton />
      <Skeleton />
    </Card.Header>
    <Card.Content flexDirection="column">
      <Skeleton />
      <div>
        {Array(3)
          .fill(0)
          .map((_, index) => (
            <Flex key={index} flexDirection="row" alignItems="start">
              <Skeleton />
              <Flex flexDirection="column">
                <Skeleton />
                <Skeleton />
              </Flex>
            </Flex>
          ))}
      </div>
    </Card.Content>
    <Card.Footer>
      <Skeleton />
    </Card.Footer>
  </Card.Root>
);
```
