## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/checkbox
```

## Usage

Import the <strong>Checkbox</strong> and piece all composition components together.

```jsx
import { Checkbox } from "@rempi-ui/checkbox";

() => (
  <Checkbox.Label>
    <Checkbox.Check />I have read and accept the terms of use
  </Checkbox.Label>
);
```

## About

Rempi-UI Checkbox component is built on top of [Radix Checkbox](https://www.radix-ui.com/primitives/docs/components/checkbox).

## API reference

### Check

It's a mix of [Radix Checkbox Root](https://www.radix-ui.com/primitives/docs/components/checkbox#root) and [Radix Checkbox Indicator](https://www.radix-ui.com/primitives/docs/components/checkbox#indicator).<br/><br/>
Contains all the parts of a checkbox. An `input` will also render when used within a `form` to ensure events propagate correctly. <br/>
Renders when the checkbox is in a checked or indeterminate state. You can style this element directly, or you can use it as a wrapper to put an icon into, or both.
