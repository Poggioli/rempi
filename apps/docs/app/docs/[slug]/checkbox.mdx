# Checkbox

A control that allows the user to toggle between checked and not checked.

<Flex style={{ gap: 8 }}>
  <RadixUI href="https://www.radix-ui.com/primitives/docs/components/checkbox" />
  <RadixAPI href="https://www.radix-ui.com/primitives/docs/components/checkbox#api-reference" />
</Flex>

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/checkbox
```

## API reference

### Check

Contains all the parts of an checkbox. <br/>
Has common <strong>Props</strong> with [Radix Checkbox Root](https://www.radix-ui.com/primitives/docs/components/checkbox#root) and [Radix Checkbox Indicator](https://www.radix-ui.com/primitives/docs/components/checkbox#indicator).

### Label

Renders an accessible label associated with controls.
Has common <strong>Props</strong> with [Rempi-UI Label](/docs/label).

  data={[

## Usage

Import the <strong>Checkbox</strong> and piece all composition components together.

```jsx
import { Checkbox } from "@rempi-ui/checkbox";

() => (
  <Checkbox.Label>
    <Checkbox.Check />I have read and accept the terms of use
  </Checkbox.Label>
);
```
