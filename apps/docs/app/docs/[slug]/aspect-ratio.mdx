# Aspect Ratio

Displays content within a desired ratio.

<Flex style={{ gap: 8 }}>
  <RadixUI href="https://www.radix-ui.com/primitives/docs/components/aspect-ratio" />
  <RadixAPI href="https://www.radix-ui.com/primitives/docs/components/aspect-ratio#api-reference" />
</Flex>

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/aspect-ratio
```

## Example

<PreviewComponent name="aspect-ratio" />

## Usage

Import the <strong>AspectRatio</strong>.

```jsx
import { AspectRatio } from "@rempi-ui/aspect-ratio";

() => (
  <AspectRatio>
    <img
      className="img-aspect-ratio"
      src="https://images.unsplash.com/photo-1535025183041-0991a977e25b?w=300&dpr=2&q=80"
      alt="Landscape photograph by Tobias Tullius"
    />
  </AspectRatio>
);
```
