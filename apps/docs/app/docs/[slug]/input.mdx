# Input

<Description text="Captures user input." />

## Example

<PreviewComponent name="input" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/input
```

## Usage

Import the <strong>Input</strong> and piece all composition components together.

```jsx
import { Input } from "@rempi-ui/input";

() => (
  <Input.Label>
    Name
    <Input.Root placeholder="Enter your name" />
  </Input.Label>
);
```
