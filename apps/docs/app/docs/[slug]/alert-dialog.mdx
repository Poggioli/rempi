## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/alert-dialog
```

## Usage

Import the <strong>AlertDialog</strong> and piece all composition components together.

```jsx
import { AlertDialog } from "@rempi-ui/alert-dialog";
import { Button } from "@rempi-ui/button";

() => (
  <AlertDialog.Root>
    <AlertDialog.Trigger>
      <Button color="error">Delete account</Button>
    </AlertDialog.Trigger>
    <AlertDialog.Portal>
      <AlertDialog.Overlay />
      <AlertDialog.Content>
        <AlertDialog.Title>Are you absolutely sure?</AlertDialog.Title>
        <AlertDialog.Description>
          This action cannot be undone. This will permanently delete your
          account and remove your data from our servers.
        </AlertDialog.Description>
        <AlertDialog.Footer>
          <AlertDialog.Cancel>
            <Button variant="text">Cancel</Button>
          </AlertDialog.Cancel>
          <AlertDialog.Action>
            <Button color="error">Yes, delete account</Button>
          </AlertDialog.Action>
        </AlertDialog.Footer>
      </AlertDialog.Content>
    </AlertDialog.Portal>
  </AlertDialog.Root>
);
```

## About

Rempi-UI Alert Dialog component is built on top [Radix Alert Dialog](https://www.radix-ui.com/primitives/docs/components/alert-dialog).

### Cancel

A button that closes the dialog. This button should be distinguished visually from `AlertDialog.Action` buttons.<br/>
For more information access [Radix Alert Dialog Cancel](https://www.radix-ui.com/primitives/docs/components/alert-dialog#cancel) documentation.

### Action

A button that closes the dialog. These buttons should be distinguished visually from the `AlertDialog.Cancel` button.<br/>
For more information access [Radix Alert Dialog Action](https://www.radix-ui.com/primitives/docs/components/alert-dialog#action) documentation.

### Title

An accessible name to be announced when the dialog is opened. Alternatively, you can provide `aria-label` or `aria-labelledby` to `AlertDialog.Content` and exclude this component.<br/>
For more information access [Radix Alert Dialog Title](https://www.radix-ui.com/primitives/docs/components/alert-dialog#title) documentation.

### Description

An accessible description to be announced when the dialog is opened. Alternatively, you can provide `aria-describedby` to `AlertDialog.Content` and exclude this component.<br/>
For more information access [Radix Alert Dialog Description](https://www.radix-ui.com/primitives/docs/components/alert-dialog#description) documentation.

### Footer

A `<div/>` that in desktop resolution is aligned to the lower right corner and in mobile resolution is aligned to the center.
