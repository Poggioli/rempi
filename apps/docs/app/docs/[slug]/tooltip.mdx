## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/tooltip
```

## Usage

Import the <strong>Tooltip</strong> and piece all composition components together.

```jsx
import { IconButton } from "@rempi-ui/button";
import { Tooltip } from "@rempi-ui/tooltip";
import { HelpCircle } from "lucide-react";

() => (
  <html>
    {...}
    <Tooltip.Provider>
      <Tooltip.Root>
        <Tooltip.Trigger>
          <IconButton
            aria-label="settings"
            variant="contained"
            shape="rounded"
            color="primary"
          >
            <HelpCircle size={16} />
          </IconButton>
        </Tooltip.Trigger>
        <Tooltip.Content>
          Add to library
          <Tooltip.Arrow />
        </Tooltip.Content>
      </Tooltip.Root>
    </Tooltip.Provider>
  </html>
);
```

## About

<strong>Rempi-UI Tooltip</strong> component is built on top of [Radix Tooltip](https://www.radix-ui.com/primitives/docs/components/tooltip).
