# Blockquote

<Description text="A block level extended quotation." />

## Example

<PreviewComponent name="blockquote" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/blockquote
```

## Usage

Import the <strong>Blockquote</strong>.

```jsx
import { Blockquote } from "@rempi-ui/blockquote";

() => <Blockquote>Hello I'm a Blockquote</Blockquote>;
```
