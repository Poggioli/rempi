# Callout

<Description text="A short message to attract user's attention." />

## Example

<PreviewComponent name="callout" />

## Installation

Install the component from your command line.

```bash
yarn add @rempi-ui/callout
```

## Usage

Import the <strong>Callout</strong> and piece all composition components together.

```jsx
import { Callout } from "@rempi-ui/callout";
import { Info } from "lucide-react";

() => (
  <Callout.Root>
    <Callout.Icon>
      <Info size={16} />
    </Callout.Icon>
    <Callout.Text>
      You will need admin privileges to install and access this application.
    </Callout.Text>
  </Callout.Root>
);
```

The <strong>Callout.Root</strong> component is built on top of [Rempi-UI Flex](/docs/components/flex), access the documentation to see all <strong>Props</strong>. <br/>
Groups Icon and Text parts. This component is based on the `<div />` element
